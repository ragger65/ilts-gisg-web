Rironseppyou Network (Rironnet)

General information

  • The Rironseppyou Network (Rironnet) is the Linux network operated by the Glacier and Ice Sheet Research Group at the Institute of Low Temperature Science, Hokkaido University.
  • The central server is the computer wwwice, which distributes the users (via NIS = Network Information Service) and the home directories (via NFS = Network File System).
  • The home directories are located at /uchi/<username> (= /home/<username>/mnthome). In addition, each user can use work directories such as /work/earth1/<username> etc.

Backups

  • A backup of all home directories /uchi/<username> (= /home/<username>/mnthome) is written every night on the external hard disk of the computer wwwice.
  • No backups of the work directories (/work/earth1/<username> etc.) are made.

Rules and hints

  • It is your own responsibility to make regular backups of your personal data
       (on memory stick, portable hard disk, DVD or CD).
          Do not rely fully on the automatic backup of the home directories.
  • Do not eat or drink in the PC room 306, and keep it clean.
  • If you end a working session, please log out and switch off the monitor, but do not switch off the computer.
  • Do not use the server wwwice for regular work.
  • Large data files must not be kept in the home directories /uchi/<username> (= /home/<username>/mnthome).
       Please use one of the work directories instead.
  • All users must be subscribed to the Rironnet mailing list in order to get security-relevant information etc.
  • If there are any problems, please contact
       Ralf Greve <greve@xxx> or Shin Sugiyama <sugishin@xxx> (xxx = lowtem.hokudai.ac.jp).

Software

  • Compilers:
    • Intel oneAPI Base & HPC Toolkit, including Fortran and C++ compilers (network license for 2 concurrent users).
    • GNU Compiler Collection (GCC), including GFortran, C and C++.
  • Python interpreter.
  • MATLAB 24.1 (R2024a) (network license for 5 concurrent users).
  • GNU Octave (free software, largely compatible to MATLAB).
  • QGIS - free and open-source geographic information system (GIS).
  • Visualization tools GMT and GrADS.
  • NetCDF - Network Common Data Form (software libraries and self-describing, machine-independent data formats for scientific data),
    Ncview - NetCDF visual browser, NCO - NetCDF Operators, CDO - Climate Data Operators.
  • Typesetting systems LaTeX (with editor kile) and LyX.
